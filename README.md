# Sistema de Gestão de Ativos

Sistema desktop para gerenciamento de ativos desenvolvido em JavaFX e SQLite.

## Tela Principal

![Telashots/tela-principal.png

## Funcionalidades

- Cadastro de ativos
- Edição de ativos
- Exclusão de ativos
- Busca automática
- Exportação CSV
- Multiempresa
- Cadastro de locais
- Filtro por local
- Controle de patrimônio
- Controle de responsáveis

## Estrutura

Empresa
↓
Localidade
↓
Ativo

## Tecnologias

- Java 17
- JavaFX
- SQLite
- JDBC
